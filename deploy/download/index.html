Install on Android

  1. Tap Download for Android above to get the .apk.
  2. Open it. Android will ask to allow “Install unknown apps” for your browser or Files app — turn it on.
  3. Tap Install, then open CatGo. Done.

Upgrading from v1.1.8 or earlier? Those builds used a different signing key — uninstall the old CatGo first, then install this one.
